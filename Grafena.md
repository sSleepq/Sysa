## Установка и настройка
### Основная машина (Debian_1)

**Установка Node Exporter**  
Для установки Node необходимо перейти на GitHub проекта и скопировать ссылку на последнюю версию приложения  
После чего выгрузить ее с репозитория и запустить для проверки  
Ссылка на проект : https://github.com/prometheus/node_exporter
```bash
    apt update
    apt upgrade
    apt install wget 
    wget https://github.com/prometheus/node_exporter/releases/download/v1.10.2/node_exporter-1.10.2.linux-amd64.tar.gz
    tar xfvz node_exporter-1.10.2.linux-amd64.tar.gz
    cd node_exporter-1.10.2.linux-amd64
    ./node_exporter 
```
Если все верно можно зайти на на веб-интерфейс по порту `9090`  

**Установка и настройка Prometheus**  
Теперь выполним установку компонента, который будет собирать метрики со всех устройст на поторых установлен Node Exporter  
По аналогии с Node переходим на гитфаб проекта и устанавливаем его  
Ссылка на проект : https://github.com/prometheus/prometheus/
```bash
    wget https://github.com/prometheus/prometheus/releases/download/v3.8.0/prometheus-3.8.0.linux-amd64.tar.gz
    tar xfvz prometheus-3.8.0.linux-amd64.tar.gz
    cd prometheus-3.8.0.linux-amd64
    ./prometheus
```
Так же проверяем работоспособоность компонента по порту `9100`  
После того как мы убедились что все работает, необходимо изменить конфигурационный файл Prometheus и указать ему откуда брать метрики (Так как Node установален у нас только локально, поэтому метрики мы будем брать по адресу `localhost:9090`)  
Для этого зайдем в файл `prometheus.yml` и изменим там строку targets  
```bash
    static_configs:
        - targets: ["localhost:9090", "localhost:9100"]
          labels:
            app: "prometheus"
```

**Добавление служб Node Exporter и Prometheus**  
Как вы могли заметить, службы мы запускали напрямую, что мешает нам запускать их одновременно и в принципе пользоваться консолью  
Для этого добавим новые службы, тем самым запустим компоненты в фоновом режиме  
Для начала переместим наши утилиты в папку `/usr/local/bin`  
```bash
    mv node_exporter-1.10.2.linux-amd64 /usr/local/bin/
    mv prometheus-3.8.0.linux-amd64 /usr/local/bin/
```
Желательно переименовать файлы, для уменьшения имени  
После необходимо создать сами службы и указать в них путь до исполняемых файлов
