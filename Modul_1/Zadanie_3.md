## 1. Учётная запись sshuser (HQ-SRV, BR-SRV)

### 1.1. Параметры

| Параметр               | Значение                  |
|------------------------|---------------------------|
| Имя пользователя       | `sshuser`                 |
| Пароль                 | `P@sswOrd`                |
| UID                    | `2026`                    |
| Права sudo             | Без ввода пароля          |

### 1.2. Порядок действий

**Шаг 1.** Создание пользователя.

```
useradd sshuser -u 2026
```

**Шаг 2.** Установка пароля.

```
passwd sshuser
```

При появлении запроса дважды ввести `P@sswOrd`.

**Шаг 3.** Добавление в группу `wheel`.

```
usermod -aG wheel sshuser
```

**Шаг 4.** Редактирование файла `/etc/sudoers`.

Команда для редактирования:

```
mcedit /etc/sudoers
```

Необходимо найти строку:

```
# %wheel ALL=(ALL:ALL) NOPASSWD: ALL
```

и убрать знак комментария `#`, приведя к виду:

```
%wheel ALL=(ALL:ALL) NOPASSWD: ALL
```

**Шаг 5.** Сохранение файла и выход из редактора.

### 1.3. Проверка

```
su sshuser
sudo ls
```

Ожидаемый результат: команда `sudo ls` выполняется без запроса пароля.

## 2. Учётная запись net_admin (HQ-RTR, BR-RTR)

### 2.1. Параметры

| Параметр               | Значение                  |
|------------------------|---------------------------|
| Имя пользователя       | `net_admin`               |
| Пароль                 | `P@ssw0rd`                |
| UID                    | `2026`                    |
| Права sudo             | Без ввода пароля          |

### 2.2. Порядок действий

**Шаг 1.** Создание пользователя с домашним каталогом.

```
useradd -m net_admin -u 2026
```

**Шаг 2.** Установка пароля.

```
passwd net_admin
```

При появлении запроса дважды ввести `P@ssw0rd`.

**Шаг 3.** Добавление в группу `sudo`.

```
usermod -aG sudo net_admin
```

**Шаг 4.** Редактирование файла `/etc/sudoers`.

Команда для редактирования:

```
nano /etc/sudoers
```

Необходимо найти строку:

```
%sudo   ALL=(ALL:ALL) ALL
```

и заменить её на:

```
%sudo   ALL=(ALL:ALL) NOPASSWD: ALL
```

**Шаг 5.** Сохранение файла и выход из редактора.

### 2.3. Проверка

```
su net_admin
sudo ls
```

Ожидаемый результат: команда `sudo ls` выполняется без запроса пароля.
